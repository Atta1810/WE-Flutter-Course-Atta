## 🚀 How to Run These Projects

If you want to try out any of the tasks locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Atta1810/WE-Flutter-Course-Atta](https://github.com/Atta1810/WE-Flutter-Course-Atta)
    ```
2.  **Navigate to the specific task folder:**
    ```bash
    cd Task_Name_Folder
    ```
3.  **Get dependencies:**
    ```bash
    flutter pub get
    ```
4.  **Run the app:**
    ```bash
    flutter run
    ```
